# KUChromeExtention
KULASISなどの京都大学学務システムの機能を拡張する非公式拡張機能です。
現在、以下の機能を実装しています。

* 時間割ページ上に教室情報を表示
* シラバス検索ページで検索対象時限の設定欄に現在の時間割を表示
* シラバス検索ページで一覧画面内で詳細を表示
